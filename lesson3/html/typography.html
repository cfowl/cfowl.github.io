Typography

There are three fonts I will use on the Weather Fortune site. These fonts are:

  • Montserrat Alternates (medium weight), a Sans-Serif font
  • Montserrat (regular weight), a Sans-Serif font
  • Inconsolata (medium weight), a Monospace font

How They Apply

Montserrat Alternates is used for all heading elements. I chose this font for all site headings because it kind of has a cloudy feel to it that I felt would go well with a the Weather Fortune site.

Inconsolata is used for the footer element. I used this monospace font to set the footer apart somewhat, and because it has an informational feel to it.

Montserrat is used for all the other text in the main element. I chose this font for all non-heading elements because it pairs well with Montserrat Alternates, has a simple, clean feel, and is easy to read. In general, sans-serif fonts work very well on the web.
